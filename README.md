## Requirements

## Getting started

### Maven dependency
```xml
<properties>
    <ruleEngine.version>0.0.1</ruleEngine.version>
</properties>

<dependencies>
    <dependency>
        <groupId>io.meshware.rule</groupId>
        <artifactId>engine-core</artifactId>
        <version>${ruleEngine.version}</version>
    </dependency>
</dependencies>
```
